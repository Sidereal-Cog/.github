# Sidereal Cog

**Engineered for the everyday**

Browser extensions that solve mundane problems. Small tools for everyday frustrations.

---

## What We Build

Chrome extensions that do one thing well:
- Manage repetitive tasks
- Organize information you reference regularly
- Fix small workflow annoyances

No bloat, no unnecessary features, no data collection. Just practical tools.

---

## Our Repositories

Each repository is a standalone extension. Browse the repos to see what's available, or check out [siderealcog.com](https://siderealcog.com) for the full catalog.

---

## Philosophy

**Start simple.** One extension, one purpose, done well.

**Stay practical.** Built to solve real problems, not chase features.

**Keep it clean.** Minimal code, maximum utility.

---

## Contributing

Found a bug? Have a suggestion? Open an issue on the relevant extension's repository.

Want to contribute code? Pull requests are welcome. Keep changes focused and in line with the extension's purpose.
